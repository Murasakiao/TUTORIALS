**Option A: Marp CLI (Command Line)**
```bash

```

**Option B: VS Code Extension**
- Install "Marp for VS Code" extension
- Right-click each markdown file → "Export to PDF"
- Or use "Marp: Export All" command

4. Verify email

### 2.2 Set Up Profile
- Add profile picture

1. Go to Settings → Payments

3. Complete identity verification

## Learning Progression

The curriculum is designed as a **cumulative stack** — each phase builds on the previous:

```
Phase 1-2    → Can use Terminal + Git
Phase 3      → Can build a basic website
Phase 4       → Can write Python scripts
Phase 5       → Can prompt AI effectively
Phase 6       → Can build a simple AI agent
Phase 7       → Can build apps with AI help
Phase 8       → Can create automations
Bonus         → Can grow a tech career + build audience
```

**Description:** Set up Visual Studio Code, the code editor used by most developers. Includes essential extensions, settings sync, and terminal integration for Mac and Windows.

**Description:** Learn Markdown in under 10 minutes. The text format developers use everywhere — READMEs, documentation, notes, and more. Write once, render anywhere.

### Tutorial 5: What is a Browser

**Description:** Understand what happens when you type a URL. Learn about HTTP requests, DNS, HTML rendering, and DevTools — the foundation of web development.

### Tutorial 6: DevTools CSS Inspection
**Title:** Inspect Any Website
**Description:** Use Chrome DevTools to inspect and modify any website in real-time. Debug CSS, test color changes, and understand how any site is built — no code required.
